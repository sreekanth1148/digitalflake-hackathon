
# Digitalflake Hackathon

This is my submission for the Digitalflake Hackathon.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## Features
- User login & authentication
- Dashboard after login
- Create and view sets

## Setup Instructions

### Backend
cd server

npm install

npm start

### Front end
cd client

npm install

npm run dev
